## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16+)
- MongoDB Atlas Account
- Google AI Studio API Key (for Yip's chat)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/NaveenSuresh07/Trellis.git
   cd Trellis
   ```

2. **Backend Setup**:
   ```bash
   cd backend
   npm install
   # Create a .env file based on the environment guide
   npm start
   ```

3. **Frontend Setup**:
   ```bash
   cd ../frontend
   npm install
   npm start
   ```
